# MS Blaze Laptop Program Proposal

Engineering Management capstone project focused on the development and launch strategy of an updated mid-tier laptop product.

## Project Overview

This project presents a complete business and technical proposal for launching the MS Blaze laptop to replace the MS Neo product line.

The project includes:
- Market analysis
- Competitor benchmarking
- Consumer requirements analysis
- Risk management
- Financial modeling
- Program lifecycle planning
- Product strategy
- Cost analysis

## Key Results

- 260,000 projected unit sales
- 30% Return on Sales (ROS)
- $40.3M projected profit
- 80% Probability of Success (POS)
